Исправьте код класса Book так, чтобы он соответствовал принципу инкапсуляции.
PS: сделайте так, чтобы свойства класса не были доступны из вне класса.

class Book {

    String title;
    String author;

}

PS: если у свойства класса не указан идентификатор доступа это означает,
что это свойство доступно для всех классов внутри пакета в котором
расположен сам класс. То есть любой класс внутри этого же пакета может
напрямую обращаться к свойствам данного класса. Этот уровень доступа
называется package level access.
