В коде ниже допущена ошибка. Исправьте её.
Вносить изменения можно только в класс ClassY.

class ClassX {
    public ClassX(int i) {
        System.out.println(1);
    }
}

class ClassY extends ClassX {
    public ClassY() {
        System.out.println(2);
    }
}

Помните о том, что если класс наследуется от другого класса, то
в его конструкторе первой строкой должен стоять вызов одного
из конструкторов родительского класса. Этот вызов может быть явным
(используя ключевое слово super), а может быть не явным. Если
в коде явно не указан вызов конструктора супер класса, то Java
во время компиляции пытается подставить первой строкой в конструктор
вызов дефолтного конструктора родительского класса (super()). Если дефолтного
конструктора в родительском классе нет, то при компиляции возникнет
ошибка (как в приведённом выше примере). Исправить такую ошибку
можно только указанием явного вызова конструктора родительского класса.
